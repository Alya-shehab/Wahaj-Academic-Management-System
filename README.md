# Wahaj Academic Management System

A Java Object-Oriented project designed to manage academic-related data in a simplified system.  
Includes project & task management, student and course records, grading features, and a secure login system.

Built as a university OOP project to apply modular design, inheritance, encapsulation, and data structures.
